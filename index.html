What is a NFT?

A NFT or non-fungible token is a type of cryptographic token in the blockchain.

It is used as a secure way to represent something unique or non-fungible.

Fungible

Fungibility is the interchangeability of a token into other tokens without the loss of intrinsic value

Bitcoin, Ethereum, and ERC-20 Tokens are considered as Fungible tokens as they can be traded or swapped with while retaining their use and value.

Non-Fungible

Non-fungible is the opposite of fungible - meaning they are unique and cannot be interchangeable.

Non fungible goods are often rare or limited in value.

ERC

ERC stands for Ethereum Request for Comments

It is an application-level standard that defines a set of functions for a token type, allowing smart contract and application interactions.

ERC-20

ERC-20 is the most commoen ERC standard being used solely in the Ethereum platform.

This is an open standard commonly used to create fungible tokens that can be traded within the Ethereum ecosystem.

ERC-721

ERC-721 is a free and open standard that defines a set of functions on how to build unique or non-fungible tokens.

NFT Marketplaces

NFT Marketplaces are where NFTs can be created or sold.

Collectibles

Collectibles is a common term used within the NFT ecosystem to represent a non-fungible token.
